📝 Note for

📋 Import dari Paste Text (copy dari Docs/Word/Email/Excel)

💡 Copy list domain dari dokumen apa saja (Word, Google Docs, Excel, email) → paste di bawah. Auto-detect format:
• TSV (tab): domain.com⇥25⇥30⇥22...
• Space: domain.com 25 30 22 ...
• List only: satu domain per baris
💾 Data tersimpan PERMANEN di browser (localStorage). Tidak hilang meski tutup browser. Hanya hilang kalau kamu Clear Browser Data atau hapus manual via 🗑/💀. Sarankan download backup JSON tiap minggu biar aman.
💡 Format per row: domain.com DA PA SS Age DR UR BL RD goodBL1, goodBL2... (kolom 9+ jadi GoodBL per-row)
💡 Kosongkan kalau tiap domain punya GoodBL berbeda (pakai per-row di text paste).
💡 Baca Clipboard: copy text di mana saja (Ctrl+C) → klik tombol → auto paste + auto import. Gak perlu paste manual.

📸 Import dari Screenshot (AI Vision)
